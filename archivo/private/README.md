# Archivo reservado

Esta carpeta contiene materiales documentales que no aparecen en la navegación pública del archivo.

El acceso se realiza mediante una clave impresa en el libro.

No se trata de ocultar información.

Se trata de recordar que **todo archivo implica un umbral**.

Durante siglos los archivos tuvieron puertas, llaves y permisos de consulta.

La red tiende a eliminar esos umbrales.

Este pequeño gesto intenta restituir uno.

---

Nota

El primer algoritmo diseñado para ser ejecutado por una máquina fue publicado por Ada Lovelace en 1843, en sus notas sobre la Máquina Analítica de Charles Babbage.

La máquina nunca llegó a construirse.

El algoritmo sí.
