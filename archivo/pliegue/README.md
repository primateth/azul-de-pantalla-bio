# Pliegue

El pliegue es una zona de ambigüedad.

Aquí se reúnen textos que no pertenecen del todo a una categoría única: filosofía híbrida, pensamiento experimental, documentos difíciles de clasificar.

El archivo no siempre puede organizar el conocimiento en líneas rectas.

A veces el pensamiento se despliega como un pliegue.
