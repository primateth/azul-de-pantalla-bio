# Técnica

La técnica no es una herramienta.

Es un entorno.

Aquí se reúnen textos sobre tecnología, sistemas, infraestructura, control algorítmico y arquitecturas de poder técnico.

La técnica ya no sirve al mundo humano.
Cada vez con más frecuencia lo organiza.

Este archivo conserva documentos que ayudan a entender esa transformación.
