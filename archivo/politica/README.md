# Política

La política fue durante siglos el arte de gobernar hombres.

Hoy es, cada vez más, la gestión de sistemas.

Este archivo reúne documentos sobre poder, administración, doctrina, gobernanza y arquitectura institucional.

No todos los textos aquí recogidos son actuales.
Algunos son antiguos porque los sistemas nuevos suelen esconderse en teorías viejas.
