# Sombra

Toda arquitectura de poder proyecta una sombra.

En ella aparecen documentos marginales, hipótesis incómodas, teorías olvidadas y textos que no suelen formar parte de la narrativa oficial.

Este archivo no afirma.

Conserva.

La sombra no siempre contiene la verdad, pero casi siempre revela lo que el discurso dominante intenta ocultar.
