# Sala fría — contenido

Este directorio reúne materiales suplementarios asociados al proyecto **Azul de Pantalla**.

No forma parte de la navegación pública del archivo.  
Se accede a él únicamente a través del umbral de la **sala fría**.

Los documentos aquí conservados amplían el registro del libro.  
No constituyen una edición crítica ni una biblioteca completa.

Funcionan como **anexos de archivo**.


## Tipos de documentos

El contenido puede incluir:

- documentos de trabajo
- referencias bibliográficas ampliadas
- notas de edición
- textos complementarios prohibidos
- materiales documentales asociados al proyecto


## Naturaleza del archivo

Los archivos no se presentan como una colección cerrada.

Este espacio funciona como una **cámara de conservación**:  
un lugar donde ciertos materiales se preservan antes de integrarse —o desaparecer— del archivo principal.


## Principio del archivo

La red es volátil.  
Los enlaces se rompen.  
Los textos se pierden.

La sala fría existe para ralentizar ese proceso.


## Nota

El primer algoritmo diseñado para ser ejecutado por una máquina fue publicado por Ada Lovelace en 1843, en sus notas sobre la Máquina Analítica de Charles Babbage.

La máquina nunca llegó a construirse.

El algoritmo sí.
