# Sala fría

Este directorio no aparece en la navegación principal del archivo.

Contiene materiales documentales asociados al proyecto **Azul de Pantalla** que requieren un pequeño umbral de acceso.

Durante siglos los archivos tuvieron puertas, permisos y llaves de consulta.

La red tiende a eliminar esos gestos.

Esta carpeta intenta satirizar de forma mínima.

---

Nota

El primer algoritmo diseñado para ser ejecutado por una máquina fue publicado por Ada Lovelace en 1843, en sus notas sobre la Máquina Analítica de Charles Babbage.

La máquina nunca llegó a construirse.

El algoritmo sí.
